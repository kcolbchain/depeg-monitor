Prices polled from Binance public API, Coinbase public API, and CoinGecko (as a DEX-weighted proxy). This dashboard is a browser-only preview of the full depeg-monitor library, which runs server-side with direct Uniswap V3 TWAP and Curve pool reads and pushes alerts to Discord/Slack/HTTP. The library and this dashboard share the same threshold semantics: bps deviation from the configured peg, warning vs critical.
